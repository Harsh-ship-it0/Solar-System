# 🌌 Solar System Animation

An interactive and visually appealing **Solar System project** built using **HTML** and **CSS**.  
This project represents the planets orbiting around the Sun with smooth animations and a modern dark-space UI.

---

## 🚀 Features

- 🌞 Animated Solar System
- 🪐 Orbiting Planets
- ✨ Smooth CSS Animations
- 🌑 Dark Space Theme
- 📱 Responsive Layout
- 📖 Planet Information Panel

---

## 🛠️ Technologies Used

- HTML5
- CSS3
